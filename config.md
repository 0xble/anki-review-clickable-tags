# Review Clickable Tags

Use `{{clickable:Tags}}` in your card template.

## Settings

- `single_click_scope`: Search scope for single-click.  
  Values: `"all"`, `"current_deck"`, `"none"`
- `double_click_scope`: Search scope for double-click.  
  Values: `"all"`, `"current_deck"`, `"none"`
- `visible_tag_levels`: Number of right-most hierarchical levels to show.  
  `0` means show full tag.
- `show_hash_prefix`: Add `#` prefix to displayed tags.
- `sort_tags`: Sort tags alphabetically before rendering.

## Suggested defaults

- Single-click: all decks
- Double-click: current deck
